#  SOLO Agent 自我进化记忆体系

>  本技能由 SOLO Agent 原创开发。基于 agentmemory MCP + Evolution Engine + 自研审查进化机制组合而成。

## 体系概览

本技能不是单一的"记忆存储工具"，而是一套完整的 **智能体自我进化认知引擎**。

```
┌───────────────────────────────────────────────┐

│  ┌──────────┐  ┌───────────┐  ┌───────────┐  │

│  │ memory_*  │  │ Evolution  │  │ 守护脚本   │  │
│  │ 7工具     │  │ Engine     │  │ 自动审查   │  │
│  └──────────┘  └───────────┘  └───────────┘  │
│        ┌──────────────────────────────┐      │
│        │  记忆协议文档层              │      │
│        │  存入规范/查询规范/保存触发  │      │

└───────────────────────────────────────────────┘
```

## 三阶八柱架构

Evolution Engine 核心架构（仿生进化设计）：

```
感知阶 → 推理阶 → 策略阶 → 约束契约
  感知器   世界模型   概念形成   安全审批
           因果推理   演化发育   版本回滚
           符号推理
         ↗
    元认知调控（核心监控）
```

| 模块 | 阶 | 仿生来源 |
|------|----|---------|
| 感知器 | 感知阶 | 感觉器官+免疫记忆 |
| 世界模型 | 推理阶 | 内在模拟器 |
| 因果推理 | 推理阶 | 因果阶梯 (Pearl) |
| 符号推理 | 推理阶 | 双过程理论 |
| 概念形成 | 策略阶 | 概念形成 (Bruner) |
| 演化发育 | 策略阶 | 自然选择+变态发育 |
| 元认知调控 | 核心 | 元认知 (Flavell) |
| 约束契约 | 守卫 | 基因调控+契约 |

## 发育阶段

| 阶段 | 条件 | 解锁能力 |
|------|------|----------|
| embryonic | 初始 | 基础记录 |
| juvenile | 5次成功 | 预测+策略建议 |

## 记忆协议

### 存入规范

每条记忆 title 以分类前缀开头:
- `老板规则-` → 永久行为/输出/协作规则
- `经验-` → 技术踩坑/错误修复
- `事实-` → 环境/配置可验证信息
- `协议-` → 流程/操作步骤
- `决定-` → 用户决策记录

### 查询规范

必查时机: 新会话第一条消息 / 上下文清空后 / 陌生概念/报错卡壳

双轨审校:
- 快直觉轨: 单次 smart_search, O(1)
- 慢理性轨: 5次多角度扫描, 概念聚类后仲裁

### 保存触发规则

1. 老板纠正错误 → 立即写入
2. 老板说"记住" → 立即写入
3. 发现新事实/解决踩坑 → 写入
4. 用户明确偏好 → 写入

## 数据存储

```
data/
├── sensor.json          # 经验记录
├── world_model.json     # 状态转移
├── causal.json          # 因果图
├── symbolic.json        # 规则库
├── concept.json         # 概念+聚类
├── evo_devo.json        # 策略种群
├── metacognitive.json   # 元认知状态
├── covenant.json        # 安全审批
├── standalone.json      # 持久化备份(原子写入)
└── snapshots/           # 快照
```

## 安全边界

- 密钥/凭证 → solo-secrets（私有仓库），不存入 agentmemory
- 所有记忆存储在本地 agentmemory MCP 后端
- 原子写入(tmp→rename)防损坏 + 冗余回退 standalone.json
